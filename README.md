# Deep_Learning_02456

This is the final project repository for the subject Deep Learning (02456)

It is organized into three main folders and the README.md file:

- "code" folder: Contains the Jupyter notebook GraphSage.ipynb, which includes the full implementation used to reproduce the results obtained in the project. It also includes Plots.ipynb, where all the figures used in the report are created.
  
- "Models" folder: It has the trained model weights produced by GraphSage.ipynb, saved as .pth files.

- Images folder: Contains all images generated in Plots.ipynb
